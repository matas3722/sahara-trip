# Pakeliui į Sacharą

Pilnas React + TypeScript + Vite projektas.

## Paleidimas kompiuteryje

```bash
npm install
npm run dev
```

## Galutinės versijos patikrinimas

```bash
npm run build
npm run preview
```

## Įkėlimas į Vercel

1. Įkelk šį aplanką į GitHub repozitoriją.
2. Vercel pasirink **Add New → Project**.
3. Importuok GitHub repozitoriją.
4. Framework preset bus aptiktas kaip **Vite**.
5. Build command: `npm run build`.
6. Output directory: `dist`.

## Duomenys

Užduočių, daiktų ir playlisto pakeitimai saugomi kiekvieno naršyklėje per `localStorage`.
